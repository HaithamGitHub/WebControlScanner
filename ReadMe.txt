to use the Web Controls scanning tool:

the soruce code is on the following github <https://github.com/HaithamGitHub/WebControlScanner.git>

1. Download the project from the github
2. import into any IDE platform eclipise or Inteliji
3. update the configuration file with your application link that need to scan < ..\AutomationControlsID\src\test\resources\Config\config.properties>
4. Run the project from IDE or by using the following command < c:\..\AutomationControlsID\mvn test >
